An Efficient 3D Deep Neural Architecture for Segmentation of Blockages in Heart Using Cardiac MRI Images

[Author names] · [Supervisor] · [Department] · [University]

Abstract

We design and simulate multiple 3D deep learning solutions for cardiac structure segmentation and heart blockage detection from cardiac MRI (CO5: design and simulation; CO14: communication). We use the ACDC dataset (Bernard et al., 2018) and implement three 3D architectures—3D U-Net (Çiçek et al., 2016), V-Net (Milletari et al., 2016), and ResAtt-3D-U-Net (Oktay et al., 2018)—with a common blockage-detection and anatomical-region pipeline. ResAtt-3D-U-Net achieved the best segmentation (Dice 0.91, IoU 0.84, accuracy 99.6%), supporting an efficient 3D deep neural architecture for cardiac segmentation.

1. Dataset

ACDC (Dataset copy 7) — Automated Cardiac Diagnosis Challenge cardiac MRI (Bernard et al., 2018). NIfTI 3D volumes; training: patients 001–100, testing: 101–150. Preprocessing: resample to 128×128×64, normalize. Ground-truth masks for LV, RV, myocardium.

2. Models (Ch. 4.2)

  • 3D U-Net: 4-level encoder–decoder, 32 base filters, skip connections (Çiçek et al., 2016).
  • V-Net: Residual blocks, 5×5×5 conv, instance norm, 16 base filters (Milletari et al., 2016).
  • ResAtt-3D-U-Net: Residual + attention gates, 16 base filters (Oktay et al., 2018).

Common post-processing: blockage detection (vessel narrowing, disconnected regions, intensity anomalies) and anatomical region identification (LV, RV, myocardium, artery).

3. Methodology (Ch. 4.1)

Design process as per objectives, specifications, requirements, constraints → literature/dataset review (ACDC) → preliminary design of multiple alternative solutions → simulation for functional verification on test set.
